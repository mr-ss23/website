Code overview#

Exam#

In the exam, you will get a copy of the following code cheat sheets:

  • Pandas cheat sheet

  • Altair marks

  • Scikit-learn cheat sheets (see Moodle)

Week 1#

First Data Analysis (Nr. 4): 💻

Note

You should be able to:

  • Import CSV-data

  • Create a crosstable

Week 2#

GitHub-Repo: umfrage

Note

You should be able to:

  • Get a data overview and show the dataframe metadata

  • Rename columns

  • Change data types to numeric, category or ordinal (using loops)

  • Recode variables (e.g. from integers to likert scale)

  • Create “quasi-metric” variables

  • Save data as CSV

Not relevant: dummy-variables

Week 3#

  • 11. Introduction to Altair (not relevant)


  • 12. Loans data

Note

You should be able to:

  • Summarize the frequencies of categorical variables

  • Show levels of categorical variables

  • Obtain a data summary for categorical variables

  • 13. Simple bar chart

  • 14. Dodged bar plot (not relevant)

  • 15. Stacked bar plot

  • 16. Standardized bar plot

  • 17. Pie chart (not relevant)

  • 18. Contingency tables

Note

You should be able to create:

  • Bar plots

  • Customized plots (change axis title, sort values, …)

  • Stacked bar plots

  • Standardized bar plots

  • Contingeny tables (crosstable) with proportions

Week 4#

  • 20. Data Analysis Case: GitHub “umfrage” (see week 2: you should have a local copy of this repo called umfrage-main)

Note

You should be able to:

  • Obtain statistical summaries of numerical and categorical data

  • Create appropriate plots and customize their axes

  • 21. Scatterplot

  • 22. Dot plot mean median and mode

  • 23. Histogram

  • 24. Variance and standard deviation (kernel density plot)

  • 25. Box Plot

  • 26. Robust statistics and transformations (not relevant)

  • 27. Comparing numerical data across groups

Note

You should be able to create:

  • Scatter plots

  • Histograms

  • Kernel density plots

  • Boxplots

  • Plots to compare numerical data across groups

Optional:

  • Mapping data
